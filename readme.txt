基于图漾双目深度摄像头的灰度自适应目标检测算法
===========================

###########简述
基于图漾双目摄像头进行采集深度图视频，利用灰度自适应算法提取人头进行客流目标检测。

###########环境要求
VS2013
opencv 3.1
cmake 2.8

###########配置
1.采集视频部分根据图漾官方文档进行配置  https://percipiodc.readthedocs.io/en/latest/getstarted/index.html#id15

2.播放视频检测部分是完整工程，需要将录制的视频放入../videos文件夹下。
